# Описание проекта

Данный проект выполнен в учебных целях и представляет собой небольшое ГИС - Django-приложение, позволяющее задавать локации на карте и хранить по каждой из них определенную текстовую информацию и изображения. Работающий пример здесь: [http://avmseven.pythonanywhere.com/](http://avmseven.pythonanywhere.com/)

# Установка

Python3 должен быть уже установлен.
Затем используйте `pip` для установки зависимостей:

```
pip install -r requirements.txt
```

# Настройка 

Основные настройки среды, должны быть заданы как переменные среды в файле .env в домашнем каталоге Django - проекта, а именно:

```
DEBUG (True/False) - вкл/выкл DEBUG режима
ALLOWED_HOSTS=127.0.0.1, localhost - при локальной разработке или адрес вашего сайта в случае хостинга на сервере провайдера.  
SECRET_KEY=строк с секретныи ключом Django  
DATABASE_URL=см. ниже
```

DATABASE_URL - строка подключения к базе данных по умолчанию, составленная в соответствии с инструкциями, указанными здесь: [https://github.com/jazzband/dj-database-url#id14](https://github.com/jazzband/dj-database-url#id14)

# Использование

После запуска приложения откройте в барузере стратовую страницу. В случае локального хостинга : [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

![image](https://user-images.githubusercontent.com/90156264/232723895-7b9045c8-c702-4f77-9f08-5a6fc40e8624.png)

Для редактирования данных перейдите в административный раздел по адресу: [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
или `адрес_вашего_хостинга/admin/` где после авторизации (возможно вам нужно будет [создать суперпользователя](https://docs.djangoproject.com/en/4.1/topics/auth/default/#creating-superusers)) окажаетесь здесь:

![image](https://user-images.githubusercontent.com/90156264/232723500-159e1f80-377e-4a2c-a1b5-51137ccff70e.png)

Перйдите на страницу списка локаций:

![image](https://user-images.githubusercontent.com/90156264/232725977-563c6999-2b53-4efb-a583-5f0067de4396.png)

Редактирование сведений о локации интуитивно понятно. В нижне части формы редактирования вы сможете загрузить новые изображения, а также изменить порядок их следования
путем перетаскивания мышкой.
